How It Works

1. User enters a message or uploads an image  
2. If image is uploaded → OCR extracts text  
3. Text is converted using TF-IDF vectorizer  
4. Naive Bayes model predicts Spam or Ham  
5. Result is displayed on web interface

6. Model Training

- Dataset: SMS Spam Collection Dataset
- Text preprocessing using NLP techniques
- Feature extraction using TF-IDF
- Classification using Naive Bayes algorithm

Performance

- Accuracy: High accuracy spam classification
- Metrics:
  - Precision
  - Recall
  - F1 Score
  

  Future Improvements

- Deep Learning model (LSTM / Transformer)

- Deployment on cloud (Render / AWS / Heroku)
